# Luke 5: Cool!

I dag må du avsløre den virkelige identiteten til den ene bakmannen. Etternavnet hans fant du forhåpentligvis i går?

Fornavnet kan du finne ved å kjøre koden under.

```js
const a = 2
const b = (!![]+[])[a]
const c = (![]+[])[a]
const d = (![]+[])[a].charCodeAt(0) - a/2
const e = String.fromCharCode(d)
const f = e+b+c
const g = f.split('').reverse().join('');
const h = g.charAt(0).toUpperCase() + g.slice(1);
console.log('Jeg vet ikke om han er ' + f + ', men hans navn er ' + h + '.')
```

Nå bør du sitte med et fullstendig navn der fornavnet består av tre bokstaver, og etternavnet består av fem bokstaver. Dessverre er ikke dette hans virkelige navn, men det er et anagram for hans virkelige navn.

Vedkommende du skal frem til har tidligere vært involvert i oppstarten av flere høyt profilerte selskaper. Et av selskapene var for eksempel både kjedelig og solgte flammekastere (som ikke var flammekastere) – pussig kombinasjon!

## Løsning
I denne oppgaven fikk vi en mappe med 500 filer, hvor hver piksel var 1x1. Navnene var `colX_rowX.png`, så oppgaven var ganske klar. Vi må sette bildene sammen ved å bruke navnet på hvert bilde. For å løse oppgaven så raskt som mulig fikk jeg ChatGPT til å skrive følgende skript:

```python
import matplotlib.pyplot as plt
from PIL import Image
import numpy as np

# Extract details of the image pieces
pieces_path = [os.path.join(extract_path, f) for f in os.listdir(extract_path) if f.endswith('.png')]
pieces_path.sort()  # Ensure pieces are in correct order

# Determine the dimensions of the grid from file naming
cols = max(int(f.split('_')[0][3:]) for f in os.listdir(extract_path)) + 1
rows = max(int(f.split('_')[1][3:-4]) for f in os.listdir(extract_path)) + 1

# Load all the image pieces and arrange them into a grid
grid = [[None for _ in range(cols)] for _ in range(rows)]

for piece in pieces_path:
    filename = os.path.basename(piece)
    col = int(filename.split('_')[0][3:])
    row = int(filename.split('_')[1][3:-4])
    grid[row][col] = Image.open(piece)

# Get the dimensions of individual tiles
tile_width, tile_height = grid[0][0].size

# Create a blank canvas for the final image
final_image = Image.new("RGB", (tile_width * cols, tile_height * rows))

# Paste each tile into the correct position on the canvas
for row_idx, row_tiles in enumerate(grid):
    for col_idx, tile in enumerate(row_tiles):
        final_image.paste(tile, (col_idx * tile_width, row_idx * tile_height))

# Display the reconstructed image
plt.figure(figsize=(10, 10))
plt.imshow(np.array(final_image))
plt.axis('off')
plt.show()
```

![alt text](/bilder/stykkevis.png)

Svaret er: `STYKKEVIS`

# Luke 12: Ikke godta hva som helst!
I forskjellige sammenhenger kan det ofte være lurt å kommunisere hva man aksepterer og hva man ikke aksepterer. Altså grensesetting.

Nettleseren du bruker, kan også sende slike signaler på dine vegne, selv om det ofte kan foregå uten at du har noe forhold til det.

[Accept-headeren](https://www.rfc-editor.org/rfc/rfc9110.html#name-accept) er en del av HTTP-protokollen, og formidler en preferanse fra klient til server, for hvilken type respons klienten aksepterer.

I dag har du muligheten til å be serveren gi deg løsningen, eller dagens kodeord, hvis du forteller at det eneste du aksepterer er solution.

/sander

## Løsning

```sh
curl -H "Accept: solution" https://kodekalender.novacare.no/luke12
```

Dette returnerte:

```html

[...]
```

## Løsning

For å løse denne oppgaven brukte jeg verktøyet overpass-turbo. Jeg lagde et lite script med informasjonen vi fikk:

```
[out:json][timeout:25];
// Definer området (Bergen, Norge)
area[name="Bergen"]->.searchArea;

// Finn museer i Bergen
(
  node["tourism"="museum"](area.searchArea);
  way["tourism"="museum"](area.searchArea);
  relation["tourism"="museum"](area.searchArea);
)->.museums;

// Finn rundkjøringer i Bergen
(
  node["highway"="mini_roundabout"](area.searchArea);
  way["junction"="roundabout"](area.searchArea);
)->.roundabouts;

// Finn museer som er innenfor 200 meter fra en rundkjøring
(
  node.museums(around.roundabouts:200);
  way.museums(around.roundabouts:200);
  relation.museums(around.roundabouts:200);
);

out body;
>;
out skel;
```

# Luke 20: SNES
Et snes er en gammel mengdeenhet som tilsvarer 20 enheter eller stykker. Ordet snes er avledet av det norrøne ordet sneis, som betyr pinne eller tynn kvist. Snes betyr egentlig det antallet av noe som er tredd inn på en pinne.

SNES, eller Super Nintendo Entertainment System, er også en spillkonsoll utformet og bygget av Nintendo på 1990-tallet.

I dag er det 20. desember. Jeg sitter innelåst i kjelleren under et stinkende museum i Bergen, på fjerde dagen, fordi jeg er på sporet av Tensinet sine skumle AI-planer.

Mye tyder på at de har hyret inn et callsenter i India for å svindle norske minstepensjonister, slik at de kan kjøpe en hel masse NVIDIA H100 GPU-er. De norske pensjonistene viste seg å ikke være så lettlurte, så nå sliter de med finansieringen.

Jeg har fått oversendt en [logg over samtaler](/filer/calls.zip) fra dette callsenteret, og du må analysere dataene og finne det telefonnummeret de har ringt flest ganger. Dagens kodeord er dette telefonnummeret.

/s


## Løsning

Filen vi fikk var en SQL database. Den inneholdt akkurat 1000 nummer, så det enkleste er å lage et like script for å finne det mest ringte nummeret:

```python
import sqlite3

db_path = 'calls.db'

conn = sqlite3.connect(db_path)
cursor = conn.cursor()

cursor.execute("""
    SELECT number, COUNT(*) as call_count
    FROM outgoing_calls
    GROUP BY number
    ORDER BY call_count DESC
    LIMIT 1;
""")
most_called_number = cursor.fetchone()

conn.close()

print(most_called_number)
>> ('55549600', 2)
```

Kodeorder er: `55549600`
